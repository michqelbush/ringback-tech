# ringback-tech
AI missed-call text-back service for local businesses, co-founder &amp; technical lead (ringback.tech)

**Live site:** [ringback.tech](https://ringback.tech)

## The problem

A lot of local businesses lose customers just from missing calls, someone calls, nobody picks up, they don't leave a voicemail, they just call the next place. Ringback texts the caller back automatically when a call gets missed, to salvage the lead and turn more lookers into buyers. 

## My role

I co-founded Ringback and do the technical side of it. I built and deployed the site, handle hosting and domain setup, and the AI-powered text response logic. My co-founder leads sales and business development.

## Why this repo has no code

The actual implementation is private since Ringback is a live business, not a demo project. This repo exists to document the project and my role in it rather than expose the product internals.

## Stack (high level)

- Static front end, deployed via Netlify
- Domain/DNS managed through IONOS
- AI-powered SMS response handling
